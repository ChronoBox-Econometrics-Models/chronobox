## Prerequisites

Before getting started, you should have:

- **Python >= 3.11** installed
- Basic familiarity with **pandas** DataFrames and Series
- A working **pip** or **conda** environment

!!! tip "New to time series?"
    If you're new to time series analysis, start with the [Quick Start](quickstart.md) for a
    hands-on introduction, then read [Core Concepts](core-concepts.md) for the theoretical
    background. The [Theory](../theory/arima-theory.md) section provides deeper mathematical
    foundations when you're ready.

## What You Can Do with ChronoBox

After completing this section, you'll be able to:

- Install and configure ChronoBox with all dependencies
- Load built-in datasets and explore time series data
- Fit ARIMA models with seasonal components
- Run diagnostic tests on model residuals
- Generate out-of-sample forecasts with confidence intervals
- Choose the right model family for your research question
